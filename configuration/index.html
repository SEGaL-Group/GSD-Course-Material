2.2 eGit

2.2.1 Configuration

  • In Eclipse open the View selection (see Figure 6)
  • Select the "Git Repositories" (see Figure 7)
  • In the "Git Repositories View" click clone repository (see Figure 8)
  • Copy the http address of your github.com repository (see Figure 9)
  • Paste the URL into the URI field and use your github credentials for Authentication (see Figure 10)
  • There hshoud ebe only the master branch selected, hit next (see Figure 11)
  • Select a directory to store the code on your harddrive and hit finish (see Figure 12)
  • Right click the repository now in the view and select "Import Projects..." (see Figure 13)
  • Select "Import existing projects" and click next (see Figure 14)
  • Select the agilefant project and click finish (see Figure 15)
  • Now the agile fant project should show up in the Package Eexplorer view (see Figure 16)
  • Next we need to configure eGit to fetch changes from the main AgilEfant repository.
  • In the "Git Repositories" view right click the Fetch line of the origin remote repository and select "Change Credentials..." (see Figure 17)
  • Select the one line in "RefSpec:" and click the "Copy" button, then the "Cancel" button (see Figure 18)
  • Right Click the "Remotes" folder and select "Create Remote..." (see Figure 19)
  • Select a name like "agilefant-main" and select "Configure fetch" and hit OK (see Figure 20)
  • Click paste (see Figure 21)
  • Ignore the warning by clicking yes (see Figure 22)
  • Go to https://github.com/SuperAmerica/agilefant and copy the "Git Read-Only" url git://github.com/SuperAmerica/agilefant.git (see Figure 23)
  • Click "Change" (see Figure 21)
  • Add the SuperArmerica agilefant git url as the URI and hit finish (see Figure 24)
  • Click "Save and Fetch" (see Figure 21)
  • You should see the master branch and simply hit OK (see Figure 25)
  • In the "Git Repositories" view you should now see two "Remotes" (see Figure 26)

2.2.2 How to fetch from the original repository

  • In the "Package Explorer" view right click the project root folder and select "Team"->"Remote"->"Fetch From..."(see Figure 27)
  • Swtich the remote repository to the agilefant-main repo and click finish(see Figure 28)

2.2.3 How to commit

  • In the "Package Explorer: view right click the files you changed or the any folder with changed files and select "Team"->"Commit" (see Figure 29)
  • Check the list of modified files and add a comment that starts with "WorkItem: xxx some comment" (without the " and where xxx is replaced by a number) and commit (Note that this commit is still only on your machine)
  • Right click the root folder and select "Team"->"Push Upstream" and click through the dialogue. (see Figure 30)

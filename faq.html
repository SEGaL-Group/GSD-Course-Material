How to install an Eclipse plug-in?

  • Go to Help->Install New Software ...
  • Add either the path to the local plug-in folder that you want to install or the url of the plug-in
  • Follow the installation dialogue.
  • Restart Eclipse.
See this page for a more detailed example.

How to commit a change?

Using RTC SCM:
  • Open the Pending Changes View.
  • Create a new ChangeSet by right-clicking the component containing the changes and select ChangeSet.
  • Drag and drop the changed files in the Changes View into the ChangeSet.
  • Right-click the ChangeSet and Related Artifacts->Associate Work Item.
  • Right-click the ChangeSet and select close.
  • Right-clich the ChangeSet and deliver.
Using eGit:
  • Select the files that have been changed.
  • Right click and select commit.
  • Add a comment of the form "WorkItem: xxx some text" without the "
  • Commit.

How to create a work item?

  • In Windows/Linux use Ctlr+n and in Mac Cmd+n.
  • Type "work item" without the " and select Work Item.
  • Fill in all Fields.
  • Save.

How to set a work item as current work?

Within Eclipse:
  • Open the work item.
  • Right click on the Work Item icon in the upper left.
  • Select Set as Current WorkItem
Alternatively:
  • Open the work item.
  • Drag the Work Item icon in the upper left to the
  • To hte field displaying the current work item (usually in the lower left).

How to link a commit to a work item?

Using eGit:
  • Add the post-commit git hook to your fork (only needs to be done once).
  • Make sure that the commit comment starts with "WorkItem: xxx: some comment" (without ") , where xxx is the work item number.
  • Push you commit to your github repository.
Using RTC:
  • Right click the change set.
  • Go to "Related Artifacts" and add related artifact.

How to open a work item?

Eclipse UI:
  • Ctrl+n
  • Select Work Item.
  • Fill in the mandatory fields.

How to comment on a work item?

Eclipse UI:
  • Open the work item.
  • Click on the Add Comment link on the lower right.
  • Write your comment.
  • Click save.

How to pull changes from forked repository

In Eclipse:
  • Open the Git Repositories View
  • Right Click your repository.
  • Select Fetch from Upstream
If you want to know more about github forks see http://help.github.com/fork-a-repo/

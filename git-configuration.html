GitHub:

  • Create GitHub Account on github.com
  • Fork https://github.com/SuperAmerica/agilefant
  • go to your for and go to Admin
  • go to Service Hooks
  • select "Post-Receive URLs
  • add http://home.segal.uvic.ca/~schadr/astc/githooks/remote-post-commit.php

Jazz.net:

  • Create jazz.net account
  • email schadr@uvic.ca your full name, account name, and email used to register
  • download RTC 3.0.1.1 (link)

Eclipse:

  • Download Eclipse 3.7.1 (link)
  • Install Eclipse
  • Unzip RTC 3.0.1.1 zip and Install RTC (like any )
  • Connect to the AgilEfant SuperAmerica repository
  • Install EGit - http://download.eclipse.org/egit/updates/site.xml
  • Open Git Repository View
    • Clone your forked repository (Use the clone button)
    • Expand repository
    • Right click Working Directory and select Import Project
  • Install mylyn
  • Install ASTC plugin
